# To-Do List Avançada

Projeto de lista de tarefas (To-Do List) desenvolvido em React, com recursos avançados como:

- Hooks personalizados (`useInput`, `useLocalStorage`)
- Context API para gerenciamento global de tarefas
- Memoization com `useMemo` e `React.memo` para otimização
- Filtros de tarefas (todas, concluídas, pendentes)
- Busca por título
- Persistência de dados no `localStorage`
- Interface responsiva e moderna

## Tecnologias utilizadas

- React 18
- JavaScript (ES6+)
- CSS moderno (Flexbox, responsividade, transições)
- LocalStorage para persistência de dados

# Como rodar o projeto localmente

1.Clonar o repositório:
git clone <URL_DO_REPOSITORIO>

2.Entrar na pasta do projeto:
cd nome-do-projeto

3.Instalar as dependências:
npm install

4.Rodar o projeto em modo de desenvolvimento:
npm run dev

5.Abrir no navegador:
O projeto estará disponível em http://localhost:5173
(ou outra porta mostrada no terminal).
